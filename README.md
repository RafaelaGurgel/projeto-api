# Consumindo API Pública com JavaScript

Este projeto é uma aplicação web simples que consome dados de uma API pública utilizando HTTP e exibe essas informações no navegador. O projeto utiliza **JSON** para manipulação de dados e implementa boas práticas no consumo de APIs.

 Funcionalidades

- Botão **"Buscar"** para realizar a requisição.
- Consumo de uma API pública (JSONPlaceholder) usando `fetch()`.
- Conversão da resposta para **JSON** com `.json()`.
- Exibição dinâmica dos resultados no navegador usando `createElement()`.
- Tratamento de erros com `try...catch` e mensagens amigáveis.

 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
